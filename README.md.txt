# Air Quality Forecasting in Delhi

This project predicts air quality levels in Delhi using machine learning techniques and historical environmental data.

## Objective
To forecast air pollution indicators such as PM2.5 and PM10.

## Methods
- Data preprocessing
- Exploratory Data Analysis
- Machine learning model training
- Model evaluation

## Tools Used
Python, Pandas, NumPy, Matplotlib, Scikit-learn
